Site TecnoLógica

Visite o Site - Clicando Aqui

O TecnoLógica é um site que desenvolvido utilizando HTML e CSS, como parte de uma aula prática da faculdade. O principal objetivo foi colocar em prática os conhecimentos adquiridos sobre estruturação de páginas web e estilização visual.

O site é dividido em três seções principais:

  • Home: exibe postagens organizadas de forma limpa e acessível, simulando um blog de tecnologia.
  • Jogos: apresenta uma galeria com imagens genéricas e textos simulados (gerados por Lorem Ipsum) para representar conteúdos relacionados ao universo dos games.
  • Smartphones: segue o mesmo modelo da aba de jogos, com imagens ilustrativas e textos fictícios, simulando um portal de notícias e análises sobre dispositivos móveis.

Mesmo sendo um projeto acadêmico e com conteúdo ainda não real, o TecnoLógica representa um exercício sólido de desenvolvimento front-end, focando em design, navegação e organização visual. Ele demonstra minha capacidade de criar interfaces funcionais e esteticamente agradáveis com as tecnologias básicas da web.


Site Gerador de Senhas

Visite o Site - Clicando Aqui

Durante um curso com foco em HTML, CSS e JavaScript, desenvolvi um site interativo chamado Gerador de Senhas, com o objetivo de aplicar conceitos de manipulação do DOM e lógica de programação em JavaScript.

O site permite ao usuário personalizar a senha que deseja gerar, oferecendo as seguintes funcionalidades:

  • Seleção de caracteres: o usuário pode escolher se a senha gerada deve conter letras maiúsculas, números e/ou símbolos especiais.
  • Controle de tamanho da senha: uma barra deslizante (slider) possibilita definir o comprimento exato da senha, de forma prática e visual.
  • Botão para copiar a senha: com um clique, a senha gerada é copiada automaticamente para a área de transferência.
  • Botão para gerar nova senha:permite refazer a senha sempre que necessário, respeitando os critérios definidos.

Mesmo sendo um projeto acadêmico e com conteúdo ainda não real, o TecnoLógica representa um exercício sólido de desenvolvimento front-end, focando em design, navegação e organização visual. Ele demonstra minha capacidade de criar interfaces funcionais e esteticamente agradáveis com as tecnologias básicas da web.
